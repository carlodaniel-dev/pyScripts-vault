# 🎥 YouTube Downloader

![Python](https://img.shields.io/badge/Python-3.x-blue?style=flat&logo=python)
![yt-dlp](https://img.shields.io/badge/yt--dlp-latest-red?style=flat)

> Download YouTube videos in MP4 format directly from the terminal.

> Descarga videos de YouTube en formato MP4 directamente desde la terminal.

---

## ⚠️ Note / Nota

> 🚧 **EN:** This script currently runs in the terminal. A graphical interface (UI) is planned for a future version.

---

## ✨ Features / Funcionalidades

- ✅ Download videos in MP4 format / Descarga videos en formato MP4
- ✅ Download by URL / Descarga por URL
- 🔜 Graphical UI / Interfaz gráfica *(coming soon / próximamente)*

---

## 📋 Requirements / Requisitos

- Python 3.x
- yt-dlp

---

## ⚙️ Installation / Instalación

**1. Clone the repository / Clona el repositorio:**
```bash
git clone https://github.com/your-username/pyScripts-vault.git
cd pyScripts-vault/youtube-downloader
```

**2. Install dependencies / Instala las dependencias:**
```bash
pip install -r requirements.txt
```

---

## 🚀 How to Use / Cómo Usar

**English:**

1. Run the script:
   ```bash
   python main.py
   ```
2. Paste the YouTube video URL when prompted
3. The video will be downloaded in MP4 format in the current folder

**Español:**

1. Pega la URL del video de YouTube cuando se te pida

2. Ejecuta el script:
   ```bash
   python main.py
   ```

3. El video se descargará en formato MP4 en la carpeta actual
